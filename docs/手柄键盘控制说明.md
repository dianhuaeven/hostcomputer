# 手柄与键盘控制说明文档

## 概述

上位机v2 支持 **键盘** 和 **手柄（XInput）** 两种输入方式控制机器人。两者共享同一套控制模式，互为备份：

- 手柄连接时，**自动禁用键盘控制**（急停快捷键始终可用）
- 手柄断开后，**自动恢复键盘控制**

系统提供两种控制模式，可随时切换：

| 模式 | 说明 |
|------|------|
| **车体模式** | 控制底盘运动（前后/转向） |
| **机械臂模式** | 控制机械臂末端位姿（笛卡尔空间） |

---

## 一、键盘控制

> 类：`KeyboardController`，发送频率 10Hz

### 1.1 车体模式

| 按键 | 功能 | 说明 |
|------|------|------|
| `W` | 前进 | `linear.x` 为当前档位正值 |
| `S` | 后退 | `linear.x` 为当前档位负值 |
| `A` | 左转 | `angular.z` 为当前档位正值 |
| `D` | 右转 | `angular.z` 为当前档位负值 |
| `1` | 切换模式 | 车体 / 机械臂切换 |
| `6` / `7` / `8` / `9` / `0` | 速度档位 | 1 档到 5 档 |
| `Space` | **急停** | 立即停止所有运动 |

- 支持**多键组合**：例如 `W+A` = 前进 + 左转（弧线运动）
- 松开所有方向键后自动发送零速度（停车）

### 1.2 机械臂模式

| 按键 | 功能 | 说明 |
|------|------|------|
| `W` / `S` | Z 轴 + / - | `twist.linear.z` |
| `A` / `D` | Y 轴 + / - | `twist.linear.y` |
| `U` / `O` | X 轴 + / - | `twist.linear.x` |
| `Q` / `E` | Roll + / - | `twist.angular.x` |
| `I` / `K` | Pitch - / + | `twist.angular.y` |
| `J` / `L` | Yaw + / - | `twist.angular.z` |
| `F` / `H` | 夹爪开 / 关 | 发布夹爪目标位置 |
| `1` | 切换模式 | 车体 / 机械臂切换 |
| `6` / `7` / `8` / `9` / `0` | 速度档位 | 1 档到 5 档 |
| `Space` | **急停** | 立即停止所有运动 |

### 1.3 默认参数

| 参数 | 默认值 |
|------|--------|
| 车体线速度 1-5 档 | 0.2 / 0.4 / 0.55 / 0.7 / 0.8 m/s |
| 车体角速度 1-5 档 | 0.4 / 0.8 / 1.0 / 1.25 / 1.5 rad/s |
| 机械臂线速度 1-5 档 | 0.04 / 0.08 / 0.10 / 0.125 / 0.15 |
| 机械臂角速度 1-5 档 | 0.2 / 0.4 / 0.55 / 0.7 / 0.8 |
| 发送间隔 | 100ms (10Hz) |

## 二、手柄控制

> 类：`HandleKey`（XInput API），轮询频率 50Hz，仅 Windows 平台

### 2.1 车体模式

| 输入 | 功能 | 说明 |
|------|------|------|
| 左摇杆 Y轴 | 前进/后退 | 线速度，最大 ±0.5 m/s |
| 右摇杆 X轴 | 左转/右转 | 角速度，最大 ±1.0 rad/s |
| `A` 按钮 | **急停** | 立即停止并发送急停指令 |

- 死区阈值：3000（摇杆原始范围 -32768 ~ 32767）
- 摇杆推得越远，速度越快（线性映射）

### 2.2 机械臂模式（末端笛卡尔空间控制）

#### 位置控制

| 输入 | 功能 | 说明 |
|------|------|------|
| 左摇杆 Y轴 | 末端前后移动 | X轴，增量 ±0.01 m |
| 左摇杆 X轴 | 末端左右移动 | Y轴，增量 ±0.01 m |
| 右摇杆 Y轴 | 末端上下移动 | Z轴，增量 ±0.01 m |

#### 姿态控制

| 输入 | 功能 | 说明 |
|------|------|------|
| 右摇杆 X轴 | 偏航 (Yaw) | 增量 ±0.05 rad |
| `LT` 扳机 | 俯仰 - (Pitch-) | 按压深度线性映射 |
| `RT` 扳机 | 俯仰 + (Pitch+) | 按压深度线性映射 |
| `LB` 肩键 | 滚转 - (Roll-) | 固定步进 -0.05 rad |
| `RB` 肩键 | 滚转 + (Roll+) | 固定步进 +0.05 rad |
| `A` 按钮 | **急停** | 立即停止并发送急停指令 |

### 2.3 模式切换（D-Pad 方向键）

| D-Pad 方向 | 切换到的模式 | modeIndex |
|------------|-------------|-----------|
| ↑ 上 | 车体模式 | 0 |
| → 右 | （预留） | 1 |
| ↓ 下 | 机械臂模式 | 2 |
| ← 左 | （预留） | 3 |

### 2.4 手柄按钮总览

| 按钮 | 车体模式 | 机械臂模式 |
|------|---------|-----------|
| A | 急停 | 急停 |
| B | — | — |
| X | — | — |
| Y | — | — |
| LB | — | 滚转 Roll- |
| RB | — | 滚转 Roll+ |
| LT | — | 俯仰 Pitch- |
| RT | — | 俯仰 Pitch+ |
| Back | — | — |
| Start | — | — |
| D-Pad | 模式切换 | 模式切换 |
| 左摇杆 | 前后速度 | 末端 X/Y 移动 |
| 右摇杆 | 左右转向 | 末端 Z 移动 + Yaw |

## 三、急停机制

急停是最高优先级操作，**不受控制模式和启用状态影响**：

| 触发方式 | 说明 |
|---------|------|
| 键盘 `Space` | 全局快捷键，任何时候可用 |
| 手柄 `A` 按钮 | 任何模式下可用 |
| UI「急停」按钮 | 界面按钮 |

急停触发后执行以下操作：
1. 通过 TCP 发送 `emergency_stop` 指令到下位机
2. 发送零速度命令（确保立即停车）
3. 重置键盘控制器状态
4. 自动切换回车体模式（安全模式）

## 四、数据流

```
┌──────────────────────┐
│  键盘按键 / 手柄摇杆  │   ← 用户输入
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│  KeyboardController  │   ← 10Hz 定时发送
│  HandleKey (XInput)  │   ← 50Hz 轮询，变化时触发
└──────────┬───────────┘
           │  Qt 信号
           ▼
┌──────────────────────┐
│     MainWindow       │   ← 根据 ControlMode 分发
│  - 车体 → velocity   │
│  - 机械臂 → joint    │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│     Controller       │   ← 业务逻辑层
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│   ROS1TcpClient      │   ← TCP JSON 发送
│   Port 9090          │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│  Intel NUC (下位机)   │   ← ROS1 执行控制
└──────────────────────┘
```

## 五、UI 显示

### 手柄状态面板 (`GamepadDisplayWidget`)

- **左右摇杆圆盘**：实时显示摇杆位置（圆点跟随移动）
- **LT/RT 扳机进度条**：竖直进度条显示扳机按压深度
- **按钮状态标签**：显示当前按下的按钮名称
- **连接状态**：绿色「已连接」/ 红色「未连接」

### 状态栏信息

- 控制模式显示：「车体运动」/「机械臂操控」
- 手柄连接状态
- 命令日志（带 500ms 节流防刷屏）
